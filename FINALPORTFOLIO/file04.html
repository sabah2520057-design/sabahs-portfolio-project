Reduction Prints

This project uses reduction linocut printing on a single block to interpret The Vessel's honeycomb structure in Manhattan through tactile, hand-crafted layers. More linoleum is carved away for each successive color, demanding precise planning from lightest to darkest shades. Starting with observational sketches, the meditative process evolved into abstraction, blending geometric complexity with vibrant form and color overlays. The final prints capture the essence of The Vessel's design while showcasing the unique textures and imperfections of hand-printing.

Duration : 2.5 weeks Type : Individual Skills Developed/ Used : Visual Composition;Reduction Printmaking;Form Abstraction;Hand Drawing & Illustration

File 02 ·
Playing Cards

This project presents a card game that teaches the anatomy of the Merriweather serif typeface through joyful illustrations of a park picnic. Each card features a letter and key anatomical clues, with the back using playful scenes like a puppy, cat, or picnic basket to visualize type features. The consistent grassy backgrounds and natural palette echo the font’s friendly tone. The game encourages players to collect cards, analyze form, and guess the typeface, blending learning with springtime charm.

Duration: 1 week Project type: Individual Skills: Illustration, Typography, Layout Design, Packaging Design
