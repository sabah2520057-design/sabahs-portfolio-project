Legacy Trunks

Legacy Trunk for Beardo reimagines men's grooming packaging as a lasting heirloom, inspired by vintage military trunks. Crafted from recyclable tin with bio-resin coating and aged patina, it features stackable compartments and NFC tag for refills. Emphasizing reuse over disposability, it transforms grooming into a sustainable ritual blending style, tradition, and sentimentality.

Duration - 1.5 weeks Project type - Individual Skills developed/used:Packaging Design;Material Exploration;Brand Translation;Spatial Thinking;Male User-Centered

Wedding Cards

This wedding invitation is made keeping elegance and the theme of romance in mind; which is being shown through its delicate floral motifs and gold accents. The ornate script and serif typefaces here are used to create a sense of luxury and tradition. This concept aims to set an enchanting yet sophisticated tone with its color palette and elements. The invite is to be printed on seed paper , ensuring that it would grow into flowers , keeping a memory of the event with the guests for a long time.

Duration : 2 days Project Type : Individual Skills developed / used : Typography;Layout Design; Color Theory ; Illustration ; Sustainable Material Knowledge;Print styles;Logo Designing; Cultural Research.
